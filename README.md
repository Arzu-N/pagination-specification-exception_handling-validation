This repository contains various Spring Boot practice examples created while learning backend development concepts.

## 📌 About
This is not a single complete project.  
It includes small, independent examples focused on different topics.

## 🔧 Topics Covered
- Pagination & Sorting
- JPA Specification (dynamic filtering)
- Validation (Hibernate Validator)
- Custom annotations
- Global Exception Handling

## 🎯 Purpose
The main purpose of this repository is to practice and understand how these concepts work in real applications.

## ⚠️ Note
The code structure may not represent a production-ready project, as it was written for learning and experimentation purposes.
